## 🪬 ~ 要件

- `Box Geometry` を利用すること
- ボックスが画面上に１００個以上描かれるようにすること
- 配置は自由
- 色や大きさなどは自由

## 🎮 ~ Getting Started

- node : v.17.0.0
- npm : 8.1.0

```
// install
npm i

// development
npm run dev

// production
npm run build

// build preview
npm run preview
```
